# Swift Server Setup Guide

## Quick Start

```bash
cd ServerProcessingServer
swift build
swift run ServerProcessingServer serve --hostname 0.0.0.0 --port 1100
```

## What This Does

✅ Uses **Swift PhotogrammetrySession** (same API as iOS)  
✅ Runs on **macOS server**  
✅ Processes images using **native Swift code**  
✅ No CLI tool needed!

## Requirements

- macOS 12.0+ (for PhotogrammetrySession API)
- Swift 5.9+
- Vapor framework (auto-installed via Package.swift)

## Advantages

- Same code/API as iOS app
- Better progress tracking
- Native Swift performance
- No external CLI dependencies
